🔧 DOM Methods Used in This App:

  • document.getElementById() - Get element by ID
  • document.createElement() - Create new HTML element
  • element.appendChild() - Add element as child
  • element.remove() - Remove element from DOM
  • element.textContent - Set/get text inside element
  • element.classList - Manage CSS classes
